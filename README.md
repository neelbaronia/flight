# How Flight Works

An interactive, first-principles introduction to flight physics for high-school learners.

The site contains three real-time 3D experiments:

- a wing lab with surface pressure, paired air packets, downwash, drag, and stall visualization
- a Wright Flyer simulation focused on takeoff and three-axis control
- a Boeing 747 simulation focused on cruise, banking, flaps, and force balance

## Run locally

```bash
npm install
npm run dev
```

Open the URL printed by Vite. The standalone art-direction study is available at `/style-directions.html`.
